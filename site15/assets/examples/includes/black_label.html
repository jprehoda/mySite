About HTML CSS JavaScript

The Chico's creative team came up with a new shopping experience, shopping by first viewing the entire outfit and then displaying each product of that outfit in a sliding side drawer for purchase. It was my task to make this a reality. I updated a pre-existing script to use jQuery deferreds on ajax calls. I also made the process of updating the experience as streamlined as possible. I turned the project from a 2 week development cycle, to being able to update the project in one day. Unfortunately, the page is no longer on the site, but I did manage to snag a link on the wayback machine.
