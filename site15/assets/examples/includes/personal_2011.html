About HTML CSS

With one personal site design and a few years of front end development experience under my belt, I want to try a design I saw online and see if I could build it from scratch. I wanted to move away from the black background/muddy color scheme I used in my first site design and create something a little classier looking. While working on a separate project at Naples Daily news I came across a jQuery plugin called beforeafter.js and wanted to work that into the design and I’m pretty happy with the outcome. All source file are available for download, so you can check my work if you’d like, or you can visit the site here.

Most of js was done with jquery plugins, so there is not much javascript to show here. All-in-all this was more of a design project more than anything else.
