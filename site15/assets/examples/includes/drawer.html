About HTML CSS JavaScript

The Soma brand was looking to expand their exposure to promotions on their site. They came to me with a request to create a drawer that slid up from the bottom of the page to display the promotions. The initial ask came before the site's move to a responsive bootstrap framework. So, after doing some research, I came across a responsive js agnostic slider which I thought was perfect for the project. Knowing that this would eventually move over to bootstrap, I coded the entire project with bootstrap classes and filled in the CSS that was missing from the site pre-bootstrap. The end result worked so well, our sister brands also decided to incorporate the new promotion tool and can be seen today on Chico's, Soma and Whitehouse Blackmarket.
